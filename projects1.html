Salesforce Data Cleanup: Part 1

Mapping Field Names of Salesforce Objects

I utilized Excel sheets containing exported Salesforce data and indicated which fields for the Contact, Program Application, and Program Engagement objects were visible on the Adult Career Training team's actual Salesforce layout.

  • Contact Object: 103 fields mapped, 523 unmatched fields, 6 duplicate fields
  • Program Application Object: 68 fields mapped, 237 unmatched fields, 8 duplicate fields
  • Program Engagement Object: 22 fields mapped, 79 unmatched fields, 0 duplicate fields

By doing this field mapping, we can identify which fields are actually showing up in the user interface our database compared to fields in the background that are not being utilized.
